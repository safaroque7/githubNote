1. After creating a github file... 
                    git init
                    git add .
                    git commit -m "first commit"
                    git branch -M main
                    git remote add origin https://github.com/safaroque7/githubNote.git 
                    git push -u origin main
2. Next time to commit 
                    git add . 
                    git commit -m "custom message"
                    git push -u origin main
3. Since add and commit are mendetory. So it can be written this way. 
                    git commit -am "custom message"
4. How many commits we did. Its figure will be shown with deatils. 
                    git log
5. To remove any files. 
                    git rm [file name with extenstion]
6. Retrieving removed file. 
                    upcomming
7. Short Status. M means after modifying it has not been uploaded to git. and ?? mean it is a new file which one has not sent to github. 
                    git status -s
