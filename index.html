১১. git clean কমান্ডের ব্যবহার। ধরুন, আপনি ‍কিছু ফাইল ক্রিয়েট করেছেন। কিন্তু সেগুলো এগুলো গিটহাবে আপলোড করেননি। সেগুলো কোন কোন ফাইল সেটা বের করা সত্যিই কঠিন। সেটা বের করার জন্য নিচের কোড। git clean -i

উপরের -i, -n এবং -f নামের কমান্ডগুলোর ব্যবহার নিচের মতো করে হবে।
git clean -i আপনাকে একটা সিরিয়াল দেখাবে। সেখানের নাম্বার অনুযায়ী কার্যাদি সম্পন্ন হবে। 

                            1: clean                2: filter by pattern    3: select by numbers
                            4: ask each             5: quit                 6: help
                            


                            1. clean               - start cleaning
                            2. filter by pattern   - exclude items from deletion
                            3. select by numbers   - select items to be deleted by numbers
                            4. ask each            - confirm each deletion (like "rm -i")
                            5. quit                - stop cleaning
                            6. help                - this screen

git clean -n ব্যবহার করলে আপনাকে দেখাবে কোন কোন ফাইলগুলো ডিলিট করা যাবে।

git clean -f কমান্ড ব্যবহার করলে কোনো দেখাদেখি নাই। কমিট করা ছাড়া সব ফাইল একটা একটা করে ডিলিট করে দেবে।
